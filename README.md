# Udemy-Clone
A fully responsive front-end clone of the Udemy learning platform built with HTML5, CSS3 (Flexbox), and Font Awesome icons. Features a modern navbar, category tabs, promotional banners, curated course cards across AI, Full-Stack, Robotics, Cybersecurity, and more, plus topic tags and a professional footer.
